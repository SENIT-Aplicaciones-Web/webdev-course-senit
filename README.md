# Fundamentos de Desarrollo Web

## Resumen del Curso

Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria a la creación de sitios web sencillos con HTML y CSS. **¡No requiere descargas ni instalaciones!** Solo necesitas abrir tu navegador web.

**Duración total**: ~60 minutos  
**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
**Prerrequisitos**: Ninguno  
**Herramientas necesarias**: Solo tu navegador web (Chrome, Firefox, Safari o Edge)  
**Editor en línea**: [Programiz HTML Online Compiler](https://www.programiz.com/html/online-compiler/) — no requiere instalación  
**Repositorio de código fuente**: [ENLACE_REPOSITORIO_GITHUB](ENLACE_REPOSITORIO_GITHUB)

## Secuencia de la Lección

### Lección 1: ¿Qué es una página web? (6 minutos)

- **Descripción**: Aprende qué es una página web, qué hace un navegador y cómo se relacionan HTML y CSS. Se explica que HTML representa el contenido y CSS representa la apariencia visual.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - Una página web se puede ver desde un navegador
  - HTML define el contenido: títulos, textos, imágenes y enlaces
  - CSS define la apariencia: colores, bordes, espacios y tamaños
  - No se necesita instalar nada para practicar
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-01-que-es-una-web/`

---

### Lección 2: Primera mini página en Programiz (6 minutos)

- **Descripción**: Conoce el editor en línea Programiz y crea tu primera mini página usando un título y un párrafo.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - Programiz permite escribir HTML y ver el resultado en el navegador
  - `<h1>` se usa para un título principal
  - `<p>` se usa para escribir un párrafo
  - El resultado cambia cuando editamos el código
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-02-primera-mini-pagina/`

---

### Lección 3: Estructura HTML básica (6 minutos)

- **Descripción**: Aprende cómo HTML organiza el contenido usando etiquetas, elementos, apertura, cierre y estructura básica de una página.
- **Enlace**: [Ver la lección](https://youtu.be/fEDYzJZ0G1k)
- **Conclusiones clave**:
  - Una etiqueta puede tener apertura, contenido y cierre
  - `<html>` contiene la página
  - `<head>` guarda información de la página
  - `<body>` contiene lo que se ve en el navegador
  - `<h1>`, `<h2>` y `<p>` ayudan a organizar textos
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-03-estructura-html-basica/`

---

### Lección 4: Listas, enlaces e imágenes (6 minutos)

- **Descripción**: Agrega elementos comunes a una página web: listas, enlaces e imágenes.
- **Enlace**: [Ver la lección](https://youtu.be/MfYqeytmvjM)
- **Conclusiones clave**:
  - `<ul>` crea una lista
  - `<li>` representa cada elemento de una lista
  - `<a>` crea un enlace
  - `href` indica el destino del enlace
  - `<img>` muestra una imagen
  - `src` indica la dirección de la imagen
  - `alt` describe la imagen con texto
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-04-listas-enlaces-imagenes/`

---

### Lección 5: Introducción a CSS — colores y fuentes (6 minutos)

- **Descripción**: Aprende qué es CSS y cómo usarlo para cambiar colores, fondos y fuentes de una página HTML.
- **Enlace**: [Ver la lección](https://youtu.be/_DvfQfhZu3A)
- **Conclusiones clave**:
  - CSS cambia la apariencia de una página
  - Una regla CSS tiene selector, propiedad y valor
  - `background-color` cambia el color de fondo
  - `color` cambia el color del texto
  - `font-family` cambia el tipo de letra
  - El archivo CSS se conecta con HTML usando `<link>`
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-05-css-colores-fuentes/`

---

### Lección 6: Tarjetas, bordes y espacios (6 minutos)

- **Descripción**: Crea una tarjeta visual usando `div`, `class`, bordes, espacios y esquinas redondeadas.
- **Enlace**: [Ver la lección](https://youtu.be/cIbDRr4By2s)
- **Conclusiones clave**:
  - `div` funciona como una caja o contenedor
  - `class` permite aplicar estilos a una parte específica
  - `border` agrega un borde
  - `border-radius` redondea las esquinas
  - `padding` agrega espacio interno
  - `margin` agrega espacio externo
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-06-tarjetas-bordes-espacios/`

---

### Lección 7: Centrar y ordenar contenido (6 minutos)

- **Descripción**: Aprende a ordenar una página usando un contenedor central, ancho máximo, centrado horizontal y alineación de texto.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - Un contenedor ayuda a ordenar el contenido
  - `max-width` limita el ancho máximo
  - `margin: auto` centra una caja horizontalmente
  - `text-align: center` centra el texto
  - `line-height` mejora la lectura de los párrafos
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-07-centrar-ordenar-contenido/`

---

### Lección 8: Botones y efecto hover (6 minutos)

- **Descripción**: Convierte un enlace en un botón visual usando CSS y agrega un efecto cuando el mouse pasa encima.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - `<a>` crea un enlace
  - `href` indica el destino del enlace
  - CSS puede hacer que un enlace parezca un botón
  - `text-decoration: none` quita el subrayado
  - `padding` agrega espacio interno
  - `border-radius` redondea esquinas
  - `:hover` aplica estilos al pasar el mouse
  - `transition` suaviza el cambio visual
- **Práctica**: [Abrir ejercicio en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-08-botones-hover/`

---

### Lección 9: Página web básica completa (8 minutos)

- **Descripción**: Integra HTML y CSS para crear una página sencilla de perfil, hobby, mascota, personaje, videojuego o deporte.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - Una página completa combina estructura HTML y estilos CSS
  - Es mejor crear primero el contenido y luego aplicar estilos
  - Se pueden combinar títulos, párrafos, imágenes, listas, tarjetas y botones
  - Una página clara usa contenedores, colores suaves y espacios adecuados
- **Proyecto**: [Abrir proyecto en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-09-pagina-basica-completa/`

---

### Lección 10: Errores comunes, buenas prácticas y reto final (8 minutos)

- **Descripción**: Revisa errores frecuentes de HTML y CSS, aprende buenas prácticas básicas y personaliza una página final.
- **Enlace**: [Ver la lección](ENLACE_YOUTUBE_NO_LISTADO)
- **Conclusiones clave**:
  - Cerrar etiquetas evita errores visuales
  - Los nombres de clase deben coincidir entre HTML y CSS
  - Los atributos deben usar comillas
  - `alt` ayuda a describir imágenes
  - Conviene cambiar una cosa a la vez y revisar el resultado
  - Un código ordenado es más fácil de corregir
- **Reto final**: [Abrir reto en Programiz](https://www.programiz.com/html/online-compiler/)
  - Código de inicio: `starter-files/lesson-10-errores-reto-final/`
